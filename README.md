# CSSAnimator

An interactive CSS keyframe animation builder with 10 presets, live preview, and CSS export.

## Features
- **10 presets** — Bounce, Spin, Pulse, Shake, Fade In, Slide In, Flip, Rubber Band, Tada, Heartbeat
- **Live preview** — animates a customizable box in real time
- **Full control** — duration, delay, timing function, iteration count, fill mode, direction
- **Custom keyframes** — edit raw `@keyframes` CSS directly
- **Generated CSS** — shows the complete copy-paste-ready CSS output
- **Customizable box** — change text, background color, text color

## Structure
```
src/animations.js   # 10 @keyframes preset definitions
src/app.js          # Apply, preview, export, controls
```
